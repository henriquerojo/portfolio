Projeto integrador desenvolvido em grupo. Consiste no sistema de uma pousada fictícia com toda documentação UML. O sistema foi desenvolvido com plataformas WEB e DESKTOP. A plataforma WEB é o site que tem toda interação com o cliente, existe um sistema de reservas e cadastro de cliente, além de recuperação de senha. O DESKTOP que consiste em outro projeto mais simples, é aonde está toda parte administrativa. Tecnologias usadas: HTML, CSS, PHP, MySQL, Bootstrap, JQuery, JavaScript, Ajax, Angular e C# (Desktop).

Projeto desenvolvido em sala de aula. Consiste em um sistema de uma churrascaria. É um sistema totalmente WEB e tem cadastro de cliente com login e senha e toda a parte administrativa de reservas, produtos, tipos e clientes. Além da parte de contato usando PHPMailer. Foi desenvolvido visando a prática do PHP e integrar o PHP com banco de dados. Tecnologias usadas: PHP, Bootstrap, HTML e MySQL.
